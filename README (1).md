# Nvidia Stock Prediction

This project uses time series analysis and machine learning techniques to predict Nvidia's stock prices. The workflow is implemented in a Jupyter Notebook and covers data preprocessing, model training, evaluation, and visualization.

## Features
- Data loading and preprocessing
- Exploratory data analysis
- Time series forecasting models
- Model evaluation metrics
- Visualization of predictions

## Requirements
- Python 3.x
- Jupyter Notebook
- pandas, numpy, matplotlib, scikit-learn (install via pip if needed)

## Usage
1. Open the notebook `Nvidia1 stock prediction.ipynb - Copy.ipynb` in Jupyter.
2. Follow the cells sequentially to run the analysis and prediction.
3. Modify parameters or models as needed for experimentation.

## Folder Structure
- `Nvidia1 stock prediction.ipynb - Copy.ipynb`: Main notebook for stock prediction
